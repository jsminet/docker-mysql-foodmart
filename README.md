
## <p align="center">Foodmart Datawarehouse on MySQL</p>
  
Getting started using a pre installed Foodmart datawarehouse
    
## 🧐 Features    
- Using docker compose version 2
- Traefik as an tcp/http edge proxy
- MySQL database
        
## 🛠️ Tech Stack
- [Traefik](https://doc.traefik.io/traefik/)
- [MySQL](https://www.mysql.com/)
    
## 🛠️ Get images
```bash
$ docker compose build
```
        
## 🧑🏻‍💻 Usage
```Bash
$ docker compose pull
$ docker compose up -d
$ docker compose down -v
```
